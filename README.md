# Installation

## UV
Create a virtual environment:
```sh
$ uv venv
```
To Activate the Virtual Environment:
```sh
$ source .venv/bin/activate
```
To exit a virtual environment:
```sh
$ deactivate
```

To update all packages:
```sh
$ uv sync --upgrade
```
